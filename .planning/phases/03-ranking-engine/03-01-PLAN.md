# Phase 03 Plan 01 - Bayesian Ranking Engine

## Objective

Implement a dedicated ranking module that scores normalized places, separates ranked from unranked results, and returns deterministic ranking metadata for later UI phases.

## Tasks

1. Create a ranking module with:
   - score calculation
   - ranked place model
   - ranking metadata model
   - full ranking result model
2. Apply deterministic ordering using score, then review-count tie-breaks, then stable fallbacks.
3. Preserve unranked places in the output with `is_ranked=False` and `rank_order=None`.
4. Add tests for formula correctness, tie-breaking, and unranked retention behavior.

## Files

- Create `ranking.py`
- Create `tests/test_ranking.py`
- Modify `app.py` to store and expose the Phase 3 ranking envelope

## Verification

- Run `pytest`
- Confirm ranking tests pass and no earlier phase regressions are introduced
