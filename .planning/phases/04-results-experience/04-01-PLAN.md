# Phase 04 Plan 01 - Results UI And Cached Search Flow

## Objective

Turn the existing retrieval and ranking envelopes into a usable results experience with cached searches, a formatted ranked table, optional unranked reveal, and a map view.

## Tasks

1. Add a results formatting module that converts ranked places into table-ready and map-ready data structures.
2. Update the Streamlit app to cache identical searches with `st.cache_data` and render the metrics, Top 20 table, unranked expander, and map view.
3. Add tests covering formatting helpers and result filtering behavior.
4. Update runtime dependencies required by the interactive map implementation.

## Files

- Create `results_view.py`
- Modify `app.py`
- Modify `requirements.txt`
- Create `tests/test_results_view.py`
